# Architecture Decision Records (ADRs)

> Key decisions with rationale, alternatives considered, and date. Immutable once accepted.

## ADR Status Values

| Status | Meaning |
|--------|---------|
| `proposed` | Under discussion |
| `accepted` | Agreed upon, in effect |
| `superseded` | Replaced by a newer ADR |
| `deprecated` | No longer relevant |
| `rejected` | Considered and declined |

## Why File ADRs?

When you revisit a decision six months later and ask "why did we choose X over Y?" — the answer is here. Not in someone's memory. Not in a Slack thread. Here.
